**Цитаты**
Цитата создается символом >:

Вложенная цитата:
>Первый уровень
>>Второй уровень

**Блок кода:**
``` python
def factorial(n):
    """Вычисляет факториал числа n."""
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))
```

| Команда Git | Функция               | Описание                              |
|:------------|:---------------------:|-------------------------------------:|
| `git init`  | Создание репозитория  | Инициализация нового репозитория      |
| `git status`| Просмотр состояния    | Показывает изменённые файлы и их статус|
| `git push`  | Отправка на GitHub    | Загружает коммиты на удалённый сервер |

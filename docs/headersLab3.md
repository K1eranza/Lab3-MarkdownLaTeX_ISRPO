#### Markdown -  это лёгкий язык разметки, который используется для оформления README-файлов, документации, отчётов и даже научных текстов.

#### Markdown прост в использовании, хорошо читается как в сыром виде, так и после рендеринга, и широко поддерживается GitHub, VS Code и многими платформами.

Применяется:
Reddit
Stack Overflow
Notion
Obsidian
GitHub
GitLab

#### Зачем он нужен разработчикам?

Разработчикам Markdown необходим для быстрого и удобного оформления текстовой информации без необходимости использования сложных инструментов или знания HTML. Он упрощает создание README-файлов для проектов, документации, ведение технических блогов и описание задач в трекерах (например, Jira, Trello).
